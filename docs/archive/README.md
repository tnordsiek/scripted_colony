# Archiv

Dieser Ordner enthaelt alte Dokumentstaende und historische Zwischeninformationen.

## Verbindlichkeit

```text
Archivdateien sind nicht implementierungsverbindlich.
Archivdateien duerfen nicht als Quelle fuer MVP-Regeln, Typen, Tests oder UI-Verhalten verwendet werden.
Bei Konflikten gelten immer die aktiven Dokumente ausserhalb von docs/archive/.
```

## Wann Archivdateien gelesen werden duerfen

Archivdateien duerfen nur verwendet werden, wenn ausdruecklich nach Historie, Rueckvergleich oder frueheren Entscheidungen gefragt wird.
